Install / Run

Prerequisites

  • Windows 10 or later (64-bit)
  • No installation required—portable executable
  • Approx. 50MB disk space

Steps

  1. Download the latest release from the Downloads page
  2. Extract the ZIP file to a folder of your choice
  3. Run CustosEye.exe
  4. The web dashboard opens automatically in your browser (typically at http://localhost:5000)

Image: dashboard-first-run.png (1200x800px, 16:9 aspect ratio) - Dashboard on first run

Configuration Options

CustosEye uses a simple JSON configuration file. By default, settings are stored in data/config.json.

Key Settings

Key Default Description
port 5000 Web dashboard port
auto_open_browser true Automatically open browser on startup
scan_interval 5 Process scan interval in seconds
enable_integrity true Enable file integrity monitoring

Configuration file location: data/config.json

Examples / Use Cases

File Integrity Monitoring

Monitor critical system files or configuration directories for unauthorized changes. CustosEye computes SHA256 hashes and alerts you when files are modified, deleted, or replaced.

Image: integrity-monitoring.png (1400x900px, 14:9 aspect ratio) - Integrity monitoring dashboard

Process Trust Scoring

Get instant visibility into running processes with trust verdicts. The local trust engine evaluates each process based on code signing, path context, network behavior, and local prevalence to flag suspicious activity.

Image: process-tree.png (1400x900px, 14:9 aspect ratio) - Process tree with trust scores

Network Activity Tracking

Monitor listening ports and active network connections system-wide. Identify processes making unexpected outbound connections or listening on unusual ports.

Image: network-activity.png (1400x900px, 14:9 aspect ratio) - Network activity dashboard
