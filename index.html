Local-first system monitoring

Monitor processes, network connections, and file integrity. Everything runs on your machine, meaning no cloud, no telemetry, no external connections.

Download for Windows

Process monitoring

Track all running processes with trust scoring. See parent relationships, memory usage, command lines, and network connections. Each process is evaluated by a trust engine that considers code signing, path context, and local prevalence.

Network monitoring

Scan active network connections to see what's listening on ports and what outbound connections processes are making. Identify unexpected network activity in real time. Export your findings in multiple formats like CSV, JSON, JSONL, or XLSX for further analysis or reporting.

File integrity monitoring

Watch specific files for changes with SHA-256 hashing or lightweight mtime+size tracking. View diffs when changes are detected, this works on any type of text files including Office documents, JSONs and PDFs.
